## How to Use

1. **Open the Web App**  
   Click [here](https://ror2-save-editor.vercel.app) to open the save editor in your browser.

2. **Upload Your Save File**  
   - Drag and drop your `savefile.xml` onto the editor, or click the upload button to select it manually.  
   - **Save File Location on PC:**  
     `C:\Program Files (x86)\Steam\userdata\[Number]\632360\remote\UserProfiles`

3. **Edit Your Save Profile**  
   - Modify survivors, items, coins, and other unlockables.  
   - Use **Unlock All** to instantly unlock everything.  
   - Changes are reflected immediately in the editor.

4. **Save Your Modified File**  
   - Click the **Save** button to download the updated `.xml` save.  
   - Replace your original save in the game folder to apply changes.
